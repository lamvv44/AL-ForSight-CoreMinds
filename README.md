# AI ForeSight

AI ForeSight is a functional demo for supporting tax, zakat, and customs auditors with company risk analysis and AI-assisted investigation summaries.

The system uses synthetic Saudi company data, a local risk scoring engine, and Gemini AI for narrative investigation output when `GEMINI_API_KEY` is available.

## Features

- Dashboard for portfolio-level risk overview.
- Company list with calculated risk scores.
- Company details page with risk indicators.
- AI investigation page powered by Gemini when configured.
- Local fallback response when no Gemini key is available or the API call fails.
- Static frontend served by FastAPI.
- Synthetic JSON dataset for demonstration.

## Gemini AI Integration

Gemini integration is implemented in `ai_service.py`.

The AI service:

- Uses `GEMINI_API_KEY` from environment variables.
- Generates the AI narrative parts of the investigation.
- Preserves the same JSON response structure expected by the frontend.
- Falls back to local response content if the API key is missing or the Gemini call fails.

The risk score is not generated by Gemini.

## Risk Engine

Risk scoring is implemented in `risk_engine.py`.

The risk engine calculates:

- Risk indicators
- Risk score
- Risk level

This file contains the business calculation logic and should not be modified unless the risk methodology itself changes.

## Database

The project uses `data.json` as a synthetic demonstration dataset.

It contains company information, monthly activity, sales, declared revenue, import values, invoice counts, and benchmark data used by the risk engine.

## API

The FastAPI backend in `main.py` exposes:

- `GET /companies`
- `GET /portfolio/summary`
- `GET /companies/{company_id}`
- `POST /analyze/{company_id}`
